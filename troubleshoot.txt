2. The Logic Fix: The IsDirty Flag

You mentioned your stats are 0 for Thrall and Sergio. This is because they aren't being "Dirty"-marked correctly.

In Controller.cs, inside SpawnNPC, please ensure you are setting both the ID and the dirty flag before registering:
C#

// Inside SpawnNPC in Controller.cs
var hotData = new EntityHotData(id);
hotData.IsDirty = true; // Make sure this is set!
_registry.RegisterStats(id, in hotData);
